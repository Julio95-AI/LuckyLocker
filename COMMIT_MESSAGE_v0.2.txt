Release v0.2.0: 新增完整部署文档体系和项目结构优化

## 🚀 主要更新

### 部署相关 (Deployment)
- 新增完整部署文档体系，支持多种云平台
- 新增 Render 平台一键部署配置
- 支持环境变量配置，适配各种云平台
- 新增 Cloudflare Worker 防休眠脚本

### 文档结构 (Documentation)
- 整理项目文档到 docs/ 文件夹
- 新增 6 份详细的部署指南文档
- 所有文档使用英文文件名（避免跨平台问题）

### 工具脚本 (Scripts)
- 移动所有脚本到 scripts/ 文件夹
- 新增项目清理脚本（Windows + Linux）

## 📝 详细变更

### 新增文件

**部署配置**:
- `render.yaml` - Render 平台自动部署配置
- `.gitignore` - Git 忽略配置（防止数据库文件误提交）

**部署文档** (docs/deployment/):
- `QUICKSTART.md` - 快速部署指南（5步完成）

- `DEPLOYMENT_COMPARISON.md` - 方案全对比表

- `DEPLOY_GUIDE.md` - 部署指南索引
- `DEPLOY.md` - 传统服务器部署（从根目录移入）

**项目文档** (docs/):
- `STRUCTURE.md` - 项目结构说明
- `WECHAT_MINIPROGRAM_DESIGN.md` - 微信小程序设计文档（重命名）

**工具脚本** (scripts/):
- `cleanup.sh` - Linux 清理脚本（从根目录移入）
