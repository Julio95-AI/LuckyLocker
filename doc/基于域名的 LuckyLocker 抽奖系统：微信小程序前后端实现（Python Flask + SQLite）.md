### 🚀 现在就可以开始了！

1. **安装依赖**：`pip install -r requirements.txt`
2. **启动服务**：双击 `start.bat`
3. **访问系统**：
   - 用户页面：`http://localhost:5000`
   - 管理后台：`http://localhost:5000/admin.html`
